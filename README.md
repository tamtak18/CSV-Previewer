# CSV-Previewer 操作マニュアル

**Version 1.0 | 製作元: takasoft**

## 1. はじめに

「CSV-Previewer」は，巨大なCSVファイルでもブラウザ上で軽快にプレビューし，精査・抽出を効率化するために開発されたGUIアプリケーションです．プロフェッショナルなダークモードと視認性の高いライトモードを兼ね備え，安定した動作と柔軟なExcel書き出しをサポートします．

## 2. 画面構成と基本デザイン

本アプリは，高精細なゲーミングPCのようなメタリック・ブラックと，鮮やかなオレンジを基調としたモダンなデザインを採用しています．

### 2.1 ヘッダーとロゴ

* **ブランドロゴ**: 画面左上に表示される「CSV-Previewer」ロゴは，takasoftの信頼性を象徴しています．
* **THEME切り替え**: 画面右上のボタンにより，周囲の環境に合わせて「ダークモード」と「ライトモード」を瞬時に切り替え可能です．

## 3. 基本操作ガイド

### 3.1 ファイルの読み込み

1. 「ファイルを選択」ボタンをクリックし，プレビューしたいCSVファイルを選択します（複数選択可）．
2. 読み込まれたファイルは「タブ」として上部に並び，クリックすることで表示を切り替えられます．

### 3.2 パスナビゲーションと階層移動

* **Path表示**: 現在表示中のファイルの所在（パス）が表示されます．
* **Backボタン**: ディレクトリ構造を遡り，ファイルの位置関係を把握するための補助として機能します．

### 3.3 データのブラウジング

* **TOP / END**: データの先頭行，または最終行へ瞬時にジャンプします．
* **PREV / NEXT**: 50行単位でのページ送りが可能です．
* **GOTO**: 指定した行番号を直接入力して「GOTO」ボタンを押すことで，任意の場所へジャンプできます．

## 4. 高度な機能

### 4.1 カラムの操作（リサイズと非表示）

* **リサイズ**: カラム間の境界線を左右にドラッグすることで，幅を自由に変更できます．長い文字列はセル内で自動的に折り返されます．
* **非表示**: カラムヘッダーをクリックすると，その列が一時的に隠されます．
* **復元**: 隠されたカラムは「HIDDEN」エリアに赤色のタグでリスト化され，クリックすることでいつでも再表示できます．

### 4.2 表内検索

* 検索窓にキーワードを入力すると，現在表示されている範囲の中で一致するセルがオレンジ色にハイライトされます．

### 4.3 Excelエクスポートと安全機能（水際対策）

用途に応じた出力範囲の選択と，ブラウザのクラッシュを防ぐ安全機能を搭載しています．

* **範囲選択**: 「表示範囲のみ（現在の50行）」または「全レコード」を選択して出力できます．
* **非表示カラムの除外**: いずれの出力時も，現在「非表示」に設定しているカラムは自動的に除外されます．
* **サイズチェック機能**:
* **500MB超**: 「全レコード」出力時に警告が表示されます．
* **1.5GB超**: ブラウザの強制終了を防ぐため，「全レコード」出力を自動的にブロックします．



## 5. ステータス情報の確認

* **RECORDS**: ファイルの全レコード数を表示します．
* **FILE SIZE**: 読み込んだファイルの物理容量を表示します．エクスポート時の判断基準として活用してください．
* **DISPLAY**: 現在表示している行の範囲（例：0 - 50）が表示されます．
