# CSV-Previewer 操作マニュアル

**Version 1.0 | 製作元: takasoft**

## 1. はじめに

「CSV-Previewer」は，巨大なCSVファイルでもブラウザ上で軽快にプレビューし，精査・抽出を効率化するために開発されたGUIアプリケーションです．プロフェッショナルなダークモードと視認性の高いライトモードを兼ね備え，Excelへのスマートな書き出しをサポートします．

## 2. 画面構成と基本デザイン

本アプリは，高精細なゲーミングPCのようなメタリック・ブラックと，鮮やかなオレンジを基調としたモダンなデザインを採用しています．

### 2.1 ヘッダーとロゴ

* **ブランドロゴ**: 画面左上に表示される「CSV-Previewer」ロゴは，takasoftの信頼性を象徴しています．
* **THEME切り替え**: 画面右上のボタンにより，周囲の環境に合わせて「ダークモード」と「ライトモード」を瞬時に切り替え可能です．

## 3. 基本操作ガイド

### 3.1 ファイルの読み込み

1. 「ファイルを選択」ボタンをクリックし，プレビューしたいCSVファイルを選択します（複数選択可）．
2. 読み込まれたファイルは「タブ」として上部に並び，クリックすることで表示を切り替えられます．

### 3.2 パスナビゲーションと階層移動

* **Path表示**: 現在表示中のファイルの所在（パス）がオレンジ色のテキストで表示されます．
* **Backボタン**: パスを遡り，上位ディレクトリの構造を視覚的に把握するための補助として機能します．

### 3.3 データのブラウジング

* **TOP / END**: データの先頭行，または最終行へ瞬時にジャンプします．
* **PREV / NEXT**: 50行単位でのページ送り，またはボタンによる500行単位の高速スキップが可能です．
* **GOTO**: 指定した行番号を直接入力して「GOTO」ボタンを押すことで，任意の場所へジャンプできます．

## 4. 高度な機能

### 4.1 カラムの操作（リサイズと非表示）

* **リサイズ**: カラム（列）間の境界線を左右にドラッグすることで，幅を自由に変更できます．長い文字列は自動的に折り返して表示されます．
* **非表示**: カラムヘッダーをホバーすると現れる「Hide」をクリックすると，その列が一時的に隠されます．
* **復元**: 隠されたカラムは「HIDDEN」エリアにリスト化され，クリックすることでいつでも再表示できます．

### 4.2 表内検索

* 検索窓にキーワードを入力すると，現在表示されている50行の中で一致するセルがオレンジ色にハイライトされます．

### 4.3 Excelエクスポート

* 「EXCEL EXPORT」ボタンを押すと，現在プレビューしている内容をExcel形式で保存します．
* **重要**: 非表示に設定しているカラムはエクスポートデータから自動的に除外されるため，必要な項目だけのレポートを即座に作成できます．

## 5. ステータス情報の確認

* **RECORDS**: 読み込んだファイルの全レコード数が表示されます．
* **DISPLAY**: 現在表示している行の範囲（例：0 - 50）が表示されます．
